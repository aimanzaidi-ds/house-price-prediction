# 🏠 House Price Prediction

This project focuses on predicting house prices using Machine Learning techniques.

## 📌 Project Overview
- Built a regression model to predict house prices
- Used Random Forest Regressor
- Dataset from Kaggle

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn

## 🔍 Steps Performed
- Data Cleaning
- Handling Missing Values
- Feature Encoding
- Model Training
- Prediction & Evaluation

## 📊 Model Used
- Random Forest Regression

## 📁 Dataset
Kaggle: House Prices - Advanced Regression Techniques

## 🚀 Output
Generated predictions and created submission file for Kaggle competition.
